# Agrinho-2026 
O TerraByte é um site interativo desenvolvido por Julia Vertuan Lourenço para o Concurso Agrinho 2026, com o tema "Como a Inteligência Artificial está revolucionando o agronegócio". Utilizando HTML5, CSS3, JavaScript e Three.js, o projeto demonstra os benefícios da IA no campo, como economia de até 40% de água, redução de 70% no uso de defensivos agrícolas e aumento de 25% na produtividade. O site conta com uma calculadora inteligente que simula o impacto financeiro e ambiental da IA, um campo 3D interativo com plantas, sensores e um drone, um mapa de calor da fazenda que mostra dados de umidade e pragas por talhão, um quiz educativo, conversor de unidades, modo claro/escuro e design responsivo. O nome TerraByte une "terra" (campo) e "byte" (tecnologia), simbolizando a fusão entre o agronegócio e a era digital. A frase de efeito do projeto é: "TerraByte: onde a terra encontra o byte – tecnologia que brota do campo."
